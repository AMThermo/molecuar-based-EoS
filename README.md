# molecuar-based-EoS

This repository contains the computer codes (Python and Fortran) and related files used to model the thermophysical properties of our advances in molecular-based equations of state at the Cohesion Laboratory (Concepción University). At the time, the folder included the following work:

- Improving Equations of State for Hard Chains Based on Thermodynamic Perturbation Theory by Juan M. Uceda and Andrés Mejía. Submitted to Chemical Physics.

- Revisiting Triplet Correlation Functions of Classical Fluids by Juan M. Uceda, Alejandro Gil-Villegas and Andrés Mejía. Submitted to the Journal of Chemical Physics.
